Welcome to str-replace.io

What is it?

str-replace.io utilises the power of the cloud to do string replacements.

Why use old, out-dated methods for string replacement when you can utilise a modern, trendy RESTful service

How do I use it

Simply POST a JSON string to 

str-replace.io/replace
containing your subject string, search string and replacement string

Request

{
  "subject":"Hello World!",
  "search":"Hello",
  "replace":"Goodbye"
}

Response

{
  "subject": "Hello World!",
  "search": "Hello",
  "replace": "Goodbye",
  "result": "Goodbye World!"
}
